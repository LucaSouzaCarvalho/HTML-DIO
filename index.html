Estrutura Básica

(voltar)

Estrutura básica do HTML é Dividida em três parter:

  • Html
  • Head
  • Body

Html onde tudo que estiver entre a tag fará parte do site, header que será o cabeçalho da página e o body que será o corpo da página.

Tags

(voltar)

As tags no HTML são utilizadas para informar ao navegador toda a estrutura do site contendo títulos, textos, imagens, links...

Textos

(voltar)

Para os textos existem diversos tipos de formatações por exemplo:

  • h1-h6
  • p
  • small
  • strong
  • u

h1-h6 são usados para títulos e subtítulos

p utilizados para criação de paragráfos

small usado para criar textos pequenos

strong utilizado para formatação de um texto para negrito

u empregado para sublinhar texto

Listas Ordenadas e não Ordenadas

(voltar)

Como o próprio nome já diz, as listas ordenadas possuem uma ordem e são numeradas, já a lista não ordenada são apenas marcadas por um marcador, exemplo:

Lista Ordenada:

  1. lista1
  2. lista2
  3. lista3

Lista não Ordenada:

  • lista1
  • lista2
  • lista3

Links
